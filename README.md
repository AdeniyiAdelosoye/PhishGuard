# PhishGuard: AI Phishing Email Detector

PhishGuard is a desktop application built in Python that helps users detect potential phishing attempts in their emails. By using a weighted scoring system and analyzing links, the application evaluates the content of an email to determine its threat level.

## 🚀 Features
* **Interactive GUI:** Easy-to-use desktop interface built with Tkinter.
* **Weighted Scoring System:** Analyzes email content for common phishing keywords and patterns.
* **Link Detection:** Identifies risky URLs or misleading hyperlinks within the text.

## 🛠️ Built With
* **Python** (Core logic)
* **Tkinter** (Graphical User Interface)
